Import last.fm play counts into a rockbox db.
```
usage: rbdbutil.py [-h] --dbdir DBDIR [--lastfm LASTFM] [--info] [--find FIND] [--list-tracks] [--import-counts]

options:
  -h, --help         show this help message and exit
  --dbdir DBDIR
  --lastfm LASTFM
  --info, -I
  --find FIND
  --list-tracks, -L
  --import-counts

```
